## IMPORT SETTINGS

After placing each sprite in Unity, configure these settings in the Inspector:

| Setting | Value |
|---------|-------|
| Texture Type | Sprite (2D and UI) |
| Sprite Mode | Single |
| Pixels Per Unit | 256 (for large textures) or 16 (for pixel art) |
| Mesh Type | Full Rect |
| Filter Mode | Trilinear (HD) or Point (pixel art) |
| Wrap Mode | Repeat (for tileable) or Clamp (for sprites) |
| Compression | None |
| Max Size | 4096 |

## BATCH WORKFLOW

1. Open Midjourney Discord
2. Copy prompt from this document
3. Generate 4 variations
4. Select best result, upscale (U1-U4)
5. Download image
6. Rename to exact filename shown above
7. Place in correct folder
8. Unity will auto-import
9. Configure import settings if needed
10. Repeat for next sprite

---

## PRIORITY ORDER

Complete sprites in this order for best results:

1. **Environment** (floors, walls, doors) - Makes biggest visual impact
2. **Characters** (wizard, knight, serf) - Player sees constantly
3. **Enemies** (ghost, skeleton, bat, spider) - Common encounters
4. **Items** (keys, food, special) - Important gameplay
5. **Props** (torch, chest, barrel) - Atmospheric detail
6. **Effects** (projectiles, slashes) - Polish

---

## TIPS FOR CONSISTENT STYLE

- Always include "game sprite" or "game asset" in prompts
- Use "--v 6" for latest Midjourney version
- Add "--tile" for seamless textures (floors, walls)
- Use "transparent background" for individual sprites
- Specify size in prompt for consistency
- Use "pixel art" for retro style or "photorealistic" for HD
